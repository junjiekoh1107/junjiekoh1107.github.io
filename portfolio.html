Python Signal Analyser

Creating a Python program that implements the Largest Series Product analysis for a given input sequence and series length. This can be used to scan encrypted signals for patterns by identifying the most "potent" clusters of digits.

The program takes in a sequence of digits and a series length, and returns the largest product of the series length of digits in the sequence. It also includes error handling for invalid inputs and edge cases.
